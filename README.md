## install php dependencies
change directory to www
```
cd www
```
then update composer with
```
composer update
```

## start project
using docker
```
docker-compose up
```
then open browser to
```
localhost:8080/
```

or, if you use LAMP Stack (or WAMP, MAMP), such as WampServer and Xampp, copy www to your platform `www` or `htdocs`
